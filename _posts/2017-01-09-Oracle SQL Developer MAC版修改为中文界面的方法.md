虽非必要，不过能改过来看着也挺顺眼的……方法如下：

打开：/Applications/SQLDeveloper 2.app/Contents/Resources/sqldeveloper/ide/bin/ide.conf 加入如下两行配置文件：

```
AddVMOption -Duser.language=zh       
AddVMOption -Duser.country=CN
```
